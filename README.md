## Project Overview
**Bug Tracker** is a **web-based bug and project management system** built using **Django** and **SQLite**.  
This system allows a single developer to manage projects and track bugs efficiently. It follows the **MVT (Model-View-Template) architecture**:  

- **Models:** Define Projects, Bugs, and Users  
- **Views:** Handle user requests, authentication, and CRUD operations  
- **Templates:** Render dynamic HTML pages using Django templates and CSS  

The application provides a responsive dashboard, intuitive interface, and real-time monitoring for effective collaboration.

---

## Features

- User Authentication (Signup, Login, Logout)  
- Add, Update, Delete Projects  
- Add, Update, Delete Bugs  
- Assign priority to bugs  
- Responsive Dashboard to monitor projects and bugs  
- Intuitive UI with Django templates and CSS
